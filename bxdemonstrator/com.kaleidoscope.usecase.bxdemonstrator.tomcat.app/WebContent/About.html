About

Demon-BX (short for Online Demonstrator for Bidirectional Transformation) is built as a part of the thesis: "A Demonstrator Framework for Consistency Management Approaches" carried out by Arjya Shankar Mishra under the supervision of Jun. Prof. Dr. Anthony Anjorin and Prof. Dr. Gregor Engels at University of Paderborn.

Technologies Used

Model: eMoflon, Java Classes

View: HTML, CSS, JavaScript, JQuery, Fabricjs

Controller: Servlet, Java Classes

Web Container: Apache Tomcat

Usage

  1. For Girl's Day Workshop conducted by Jun. Prof. Dr. Anthony Anjorin on 27th April 2017.

  2. In a Lecture in Universiy of Paderborn
